# Duplicate Tab Killer

A lightweight Chrome extension for tab organization and cleanup.

- **Find & close duplicates** — live count on the toolbar, one-click close
  (keeps the oldest copy), optional auto-close, and "jump to the existing tab"
  when you open a page you already have.
- **Sessions** — group tabs you use together into named Chrome tab groups you
  can start, manage (drag tabs in / between sessions), and close all at once,
  with an optional auto-close time.
- **Stay tidy** — optional reminder for tabs idle 3+ hours, and a
  "Recently closed" log.

Everything runs locally. No accounts, no tracking, no data leaves your browser.

## Install (development)
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. **Load unpacked** → select this folder
